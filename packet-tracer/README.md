# Cisco Packet Tracer Lab

## What is this?
A network topology built in Cisco Packet Tracer demonstrating core networking concepts.

## Concepts Implemented
- VLANs and segmentation
- Router-on-a-Stick
- Sub-interfaces
- Trunk ports
- Access Control Lists (ACLs)
- DHCP server
- NAT overload (PAT)
- Static routing
- GRE tunneling
- SSH security
- Troubleshooting methodology
